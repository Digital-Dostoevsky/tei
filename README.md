# Digital Dostoevsky

[Digital Dostoevsky](https://digitaldostoevsky.com) is a computational text analysis project on a corpus of 5 novels and two novellas by Fyodor Dostoevsky. It is a digital humanities project which emerges out of our long-standing interest in traditional philological analysis. We are excited by how digital approaches such as TEI encoding, machine reading, and natural language processing can help to answer questions about the deep structure of Dostoevsky’s novels, questions about speech, character, space, temporality, affect, and fictionality, among other areas.
